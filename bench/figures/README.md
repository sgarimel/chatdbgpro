# Paper Figures

All visualizations for the ChatDBG-Pro paper. Generated by `generate_all.py`.

## Quick Start

```bash
# Generate everything
python bench/figures/generate_all.py

# Generate only one corpus
python bench/figures/generate_all.py --only synth
python bench/figures/generate_all.py --only bugbench
python bench/figures/generate_all.py --only tools
```

Requirements: `matplotlib`, `numpy`, `pandas`, `scipy` (optional, for Spearman r).

## Data Sources

| Data | Directory | Runs | Notes |
|------|-----------|------|-------|
| Synthetic T1 (bash) | `results/synth-8model-sweep/` (tier1 prefix) | 200 | 25 cases × 8 models |
| Synthetic T2 (bash+gdb) | `results/synth-8model-sweep/` (tier2 prefix) | 200 | 25 cases × 8 models |
| Synthetic T3 (ChatDBG) | `results/synth-t3-unfenced/` | 169 | 21 cases × 8 models, unfenced/unsafe=true |
| Synthetic T3+CMW | `results/synth-t3-unfenced-cmw/` | 120 | CMW scores from `collect.json` |
| BugBench T1 | `results/bugbench-t1/` | 20 | 4 cases × 5 runs (4 models + combo) |
| BugBench T2 | `results/bugbench-t2/` | 20 | same |
| BugBench T3 | `results/bugbench-t3/` | 20 | same |

**Important:** The old fenced T3 data (168 runs in `synth-8model-sweep/` with tier3 prefix) used `unsafe=false` which blocked most GDB commands. Do NOT use it for T3 analysis. Use `synth-t3-unfenced/` instead.

## Legacy Figure Locations

These older figures are superseded by this directory but kept for reference:

| Location | What | Status |
|----------|------|--------|
| `bench/analysis/figs/` | Cross-tier heatmaps, CMW, tool usage (from `visualize_cross_tier.py`) | Superseded — used old fenced T3 + old CMW data |
| `bench/analysis_artifacts/figs/` | Early harness diagnostics, BugsCPP validity, tool_cmd plots | Partially superseded — tool_cmd plots replaced here |

### How the legacy figures were generated

**`bench/analysis/figs/`** — Run from repo root:
```bash
python bench/analysis/visualize_cross_tier.py   # cross-tier bars, heatmaps, CMW, tool usage
python bench/analysis/visualize_synth.py         # T3-only synth heatmap + per-axis
python bench/analysis/visualize_final.py         # combined final figures
python bench/analysis/visualize_full_table.py    # full markdown table
```
These scripts read from `synth-8model-sweep/` (including old fenced T3) and `synth-cmw-t3-sweep/` (archived old CMW). They will produce correct T1/T2 figures but stale T3/CMW figures.

**`bench/analysis_artifacts/figs/`** — Run from repo root:
```bash
python bench/analysis_artifacts/make_figures.py       # 7 diagnostic figures (harness validity, BugsCPP binaries, etc.)
python bench/analysis_artifacts/heatmap.py             # 19-case T3 heatmap (old 4-model data)
python bench/analysis_artifacts/heatmap_real.py         # judge-scored heatmap
python bench/analysis_artifacts/strong_score.py         # anchor-based heuristic scoring
python bench/analysis_artifacts/yara_pilot_figures.py   # YARA pilot results
python bench/analysis_artifacts/build_cross_tier_pdf.py # multi-page PDF report
python bench/analysis_artifacts/cost_enrich.py          # cost/enrichment analysis
```
These read from `all_runs.csv` / `agent_runs.csv` (early data) and various result dirs. The `make_figures.py` harness diagnostics (01-07) are still relevant for the BugsCPP failure analysis.
